# Projet NoSQL et les tps

> pip install -r requirements.txt

## Windows

### Créer l'environnement
> py -3 -m venv venv
### Activer l'environnement
> venv\Scripts\activate
### Lancer le serveur
> py -3 app.py
<br>

## Mac OS

### Créer l'environnement
> python3 -m venv venv
### Activer l'environnement
> . venv/bin/activate
### Lancer le serveur
> python3 app.py
