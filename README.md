# Projet NoSQL et les tps

## Windows

### Créer l'environnement
> py -3 -m venv venv
### Activer l'environnement
> venv\Scripts\activate
### Installer les packages
> pip install -r requirements.txt
### Lancer le serveur
> py -3 app.py
<br>

## Mac OS

### Créer l'environnement
> python3 -m venv venv
### Activer l'environnement
> . venv/bin/activate
### Installer les packages
> pip install -r requirements.txt
### Lancer le serveur
> python3 app.py
